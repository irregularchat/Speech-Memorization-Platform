Setup

For full functionality, configure API keys in your .env file:

  • VITE_OPENAI_API_KEY - For Whisper and AI coaching
  • VITE_GOOGLE_CLOUD_API_KEY - For Google Speech recognition
  • VITE_AZURE_SPEECH_KEY - For Azure Speech Services
